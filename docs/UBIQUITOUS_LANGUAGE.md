### Process
A running program instance on the operating system, identified by a Process ID (PID).

### Process ID (PID)
A unique numerical identifier assigned by the operating system to each running process.

### Process Name
The executable name or command that started the process.

### Process Command Line
The full command line used to start the process, including arguments.

### Process Environment
The complete set of environment variables available to a specific process at runtime.

### Process Environment Variable
An environment variable that exists within the context of a specific process, potentially different from system or user defaults.

### Process Investigation
The act of examining running processes to discover their environment variable configurations.

### Environment Inheritance
How processes inherit environment variables from their parent processes and system defaults, with potential overrides.

### Process Snapshot
A point-in-time capture of process information and environment state.

### Process Lifetime
The duration a process runs, during which its environment may change.
